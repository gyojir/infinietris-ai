# ∞etris

Program name: `infinietris`

底が抜けている以外は普通のテトリスです。

## ルール

- 左右移動、回転、ソフトドロップ、ハードドロップあり
- ライン消去あり
- スコアはありません
- 落としたテトリミノ数に応じて落下速度が上がります
- **床だけが存在しません**
  - ミノは最下段を抜けると、そのまま画面外へ落下して消えます
  - 画面外へ完全に出た瞬間、固定されずに次のミノが出現します
  - 既に積まれているブロックに接触したときだけ固定されます
- 新しいミノの出現位置が埋まっていたらゲームオーバーです

## 起動方法

依存関係はありません。`index.html` をブラウザで開くだけで遊べます。

## 操作

### PC

- `←` / `→` : 移動
- `↑` / `X` : 右回転
- `Z` : 左回転
- `↓` : ソフトドロップ
- `Space` : ハードドロップ
- `C` : ホールド
- `P` : 一時停止

### スマホ操作

- 盤面をタップ : 回転
- 左右にスワイプ : 移動
- 下にスワイプ : ソフトドロップ

## メモ

通常のテトリスとの差分は「床判定をなくし、下端の外へ落ちたミノを消して次ミノへ進める」「落としたミノ数で速度が上がる」「スコアがない」点です。
